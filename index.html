Spring Web Services 旨在促进契约优先的 SOAP 服务开发，允许使用多种操作 XML 有效负载的方法之一来创建灵活的 Web 服务。

Spring Data Relational

Spring Data Relational 是 Spring Data 系列的一部分，可以轻松实现 SQL 数据库的存储库。该模块处理对基于 JDBC 和 R2DBC 的数据访问层的增强支持。
